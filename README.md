# prototype-hris
Prototype untuk aplikasi simple HRIS
Untuk fitur-fitur yang ada pada aplikasi ini yaitu :

1. Pembuatan Employee Baru
2. Pembuatan User Baru (Harus dibuat Employee nya terlebih dahulu)
3. Absensi untuk Clock In dan Clock Out
4. Melihat data history absensi
5. Melihat data history absensi subordinate (apabila memiliki subordinate)
6. Melihat nama Employee subordinate (apabila memiliki subordinate)
